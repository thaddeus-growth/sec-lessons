# Leçons SEC — Version Française

> **🌐 Bienvenue / Welcome / Willkommen / Bienvenido / ようこそ / 환영합니다 / 欢迎**

Ceci est la **version française** d'un cours en 9 leçons sur la surveillance des entreprises publiques américaines via leurs dépôts SEC (10-K, 10-Q, 8-K, Form 4, 13F, 13D/G) pour l'intelligence concurrentielle.

## Comment utiliser ce cours

1. **Commencez ici**: [`lessons/0001-sec-filing-101.html`](./lessons/0001-sec-filing-101.html) — introduction aux 10-K, 10-Q, 8-K
2. **Suivez l'ordre recommandé**: 0001 → 0002 → 0003 → 0004 → 0005 → 0006 → 0007 → 0008 → 0009
3. **Ayez l'aide-mémoire à portée de main**: [`reference/sec-form-types.html`](./reference/sec-form-types.html) est imprimable
4. **Ne sautez pas les quiz** — ils sont conçus pour faire ressortir les erreurs de jugement courantes

## À propos de ce cours

- **Objectif**: Enseigner les **connaissances du domaine** pour l'utilisation des divulgations d'entreprises publiques américaines à des fins d'intelligence concurrentielle
- **Public**: Analystes en investissement stratégique, équipes de due diligence, chercheurs M&A, toute personne surveillant des concurrents d'entreprises publiques
- **Format**: 9 fichiers HTML autonomes, ~10-15 min chacun, 3 questions de quiz par leçon
- **Exemples**: Carvana (CVNA), Carmax (KMX), AutoNation (AN), Uxin (UXIN), AutoHome (ATHM)
