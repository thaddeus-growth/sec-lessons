# SEC-Lektionen — Deutsche Version

> **🌐 Willkommen / Welcome / Bienvenue / Bienvenido / ようこそ / 환영합니다 / 欢迎**

Dies ist die **deutsche Version** eines 9-teiligen Kurses zur Überwachung US-amerikanischer börsennotierter Unternehmen anhand ihrer SEC-Einreichungen (10-K, 10-Q, 8-K, Form 4, 13F, 13D/G) für Competitive Intelligence.

## Wie man diesen Kurs nutzt

1. **Hier starten**: [`lessons/0001-sec-filing-101.html`](./lessons/0001-sec-filing-101.html) — Einführung in 10-K, 10-Q, 8-K
2. **Empfohlene Reihenfolge**: 0001 → 0002 → 0003 → 0004 → 0005 → 0006 → 0007 → 0008 → 0009
3. **Spickzettel bereithalten**: [`reference/sec-form-types.html`](./reference/sec-form-types.html) ist druckbar
4. **Quizfragen nicht überspringen** — sie sind darauf ausgelegt, häufige Fehleinschätzungen aufzudecken

## Über diesen Kurs

- **Ziel**: Vermittlung des **Domänenwissens** für die Nutzung von SEC-Einreichungen für Competitive Intelligence
- **Zielgruppe**: Strategische Investmentanalysten, Due-Diligence-Teams, M&A-Forscher, alle, die börsennotierte Wettbewerber überwachen
- **Format**: 9 in sich geschlossene HTML-Dateien, ~10-15 Min. pro Lektion, 3 Quizfragen pro Lektion
- **Beispiele**: Carvana (CVNA), Carmax (KMX), AutoNation (AN), Uxin (UXIN), AutoHome (ATHM)
