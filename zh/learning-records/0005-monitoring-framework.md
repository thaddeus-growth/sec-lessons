## Key insights

1. **竞品监控四层次**: 财务报表(what) → 事件驱动(when what) → 管理层信号(why) → 跨公司比较(so what)。系统做前三个，人做第四个。
2. **5 家竞品的选取逻辑** — 每家的定位不同：
   - Carvana: 验证"纯线上模式"是否成立
   - Carmax: 线下最优模式的效率基准
   - AutoNation: 产业链上游（新车→二手车）的观察窗口
   - 优信: 中国市场特殊问题的样本
   - 汽车之家: 行业流量的晴雨表
3. **跨公司交叉验证比单公司趋势分析更有价值** — 信号矩阵：当 A 公司的 X 指标变化 + B 公司的 Y 指标同时变化 → 可得出结论 Z
4. **反身性视角**: 你们的 filing 也在被竞品监控——每写一份 filing 都要想"对手读了会做什么"
5. **系统的终极价值**: 把原始数据变成结构化情报 → 人把情报变成对自己公司的战略判断
