## Key insights

1. **Section 16 的适用边界**：只覆盖持有<strong>美国注册证券</strong>的内部人。中国 ADR（UXIN/ATHM）的高管通常持 VIE 层面权益 → 不适用。所以"无数据" ≠ "采集失败"，是结构性的"不适用"。
2. **三张表的区别**：Form 3（首次声明，一次性）/ Form 4（每次交易，2 个工作日，最重要）/ Form 5（年度汇总，豁免交易）。
3. **Table I vs Table II**：Table I = 直接持股；Table II = 期权/衍生品。M（行权）+ F（交税卖出）= 配套操作，不是看空信号。
4. **交易代码的权重设计**：
   - P（主动买入）= 1.0（强信号）
   - S（主动卖出）= 1.0
   - F（交税卖出）= 0.2（保留微弱信号）
   - G（赠与）= 0.3
   - A（授予）= 0.0（不算主动交易）
   - M（行权）= 0.0（只看向行权后的卖出）
5. **角色权重**：CEO/CFO = 2.0（最敏感）/ 10% Owner = 1.5 / Director = 1.0 / EVP/SVP/VP = 0.8。
6. **10b5-1 计划是最大的误判源**：高管预设的交易计划会产生大量配对 Code F/S。**判断时必须看 10b5-1 计划的备案日**（在 8-K Item 8.01 或 DEF 14A 里），不是看交易本身。
7. **Form 144 是"预告片"**：事前通知，不是事后披露。"计划在 90 天内卖出"≠"已经卖了"。多份 Form 144 连续提交 = 持续减持信号。
8. **中国 ADR 异常告警规则**：如果 UXIN/ATHM 突然出现 Form 4 → 这是极重要信号（罕见事件），系统应触发红色预警。
