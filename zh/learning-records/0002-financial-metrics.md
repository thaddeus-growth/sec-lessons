## Key insights

1. **收入结构比总收入更重要** — 零售 vs 批发 vs 金融的占比变化往往揭示业务健康度的转变。批发占比上升 = 零售端出问题。
2. **毛利率是二手车行业的圣杯** — 反映单车买卖差价的真实水平，不受一次性费用干扰。行业正常范围 15-20%。
3. **SG&A/收入比是运营效率的核心指标** — 下降说明规模效应在兑现，上升说明在烧钱换增长。
4. **库存周转对二手车公司比一般零售更重要** — 车会贬值，库存积压 = 被迫降价 = 毛利率崩塌。
5. **资产负债表的危险信号** — 现金/季度收入 < 0.3、债务增速 > 收入增速 → 流动性风险。
