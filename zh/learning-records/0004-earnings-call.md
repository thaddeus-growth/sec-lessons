# Learning Record 0004 — Earnings Call 的战略价值

**Date**: 2026-06-16
**Lesson**: 0004-earnings-call.html
**Status**: 已完成

## Key insights

1. **Prepared Remarks 是排练过的表演，Q&A 才是没有剧本的真实对话**
2. **Guidance 的措辞变化比数字变化更早** — "有信心"→"谨慎"往往先于数字下调 1-2 个季度
3. **撤回指引 (guidance withdrawal) 比坏数字更可怕** — 说明管理层自己也看不清未来（visibility = 0）
4. **分析师问什么比管理层答什么更能说明市场在担心什么**
5. **"整体基调"是最被低估的判断** — 数字没问题但"听上去怪怪的"→ 相信直觉

## Reading order for EC summary (5 min)

1. 财务亮点 (30s) → 2. 指引变化 (1min) → 3. Q&A 关键问答 (2min) → 4. 风险提示 (1min) → 5. 整体判断 (30s)
