# SEC 课程 — 中文版

> **🌐 欢迎 / Welcome / Bienvenue / Willkommen / Bienvenido / ようこそ / 환영합니다**

这是 **9 课系列** 的中文版，教你如何通过 SEC 披露文件（10-K, 10-Q, 8-K, Form 4, 13F, 13D/G）监控美国上市公司，用于战略投资和竞品分析。

## 怎么用这个课程

1. **从这里开始**：[`lessons/0001-sec-filing-101.html`](./lessons/0001-sec-filing-101.html) — 10-K, 10-Q, 8-K 入门
2. **按推荐顺序**：0001 → 0002 → 0003 → 0004 → 0005 → 0006 → 0007 → 0008 → 0009
3. **把速查表放在手边**：[`reference/sec-form-types.html`](./reference/sec-form-types.html) 可以打印
4. **不要跳过 quiz** — quiz 是为了暴露常见判断错误设计的

## 关于这个课程

- **目标**：教授**领域知识**——怎么用美国上市公司的披露做竞品分析
- **受众**：战略投资分析师、尽调团队、M&A 研究员、监控上市公司竞争对手的人
- **形式**：9 个独立 HTML 文件，每课 10-15 分钟，每课 3 道 quiz
- **实例**：Carvana (CVNA), Carmax (KMX), AutoNation (AN), 优信 (UXIN), 汽车之家 (ATHM)
- **协议**：CC BY 4.0 — 分享、改编、署名
