# Resources

## Primary (highest trust, start here)

### SEC 官方
- [SEC EDGAR — How to Research Public Companies](https://www.sec.gov/edgar/about) — SEC 官方的 EDGAR 入门指南
- [SEC Beginners Guide to Financial Statements](https://www.sec.gov/about/reports-publications/investor-publications/beginners-guide-financial-statements) — SEC 写的财报阅读入门（面向非财务背景）
- [EDGAR Form Types](https://www.sec.gov/files/forms-listed.pdf) — 所有 SEC form 类型的完整列表
- [How to Read a 10-K](https://www.sec.gov/files/reada10k.pdf) — SEC 官方出品的 10-K 阅读指南
- [How to Read an 8-K](https://www.sec.gov/files/reada8k.pdf) — SEC 官方出品的 8-K 阅读指南

### 战略投资 / 竞品分析
- [CFI — Competitor Analysis Framework](https://corporatefinanceinstitute.com/resources/management/competitor-analysis/) — 竞品分析的经典框架
- [Aswath Damodaran — 估值与财报分析课程](https://pages.stern.nyu.edu/~adamodar/) — 华尔街最著名的估值教授，所有课件免费
- [Investopedia — 财报分析入门](https://www.investopedia.com/articles/basics/06/financialreporting.asp) — 比 SEC 官方指南更通俗

### 二手车行业专用
- [Carvana Investor Relations](https://investors.carvana.com/) — 直接看 Carvana 的 IR 页面，理解上市公司如何与投资者沟通
- [NADA — Auto Retail Industry Data](https://www.nada.org/nada/industry-analysis) — 美国汽车经销商协会的行业数据

## Secondary (useful depth)

- [XBRL US — What is XBRL?](https://xbrl.us/home/consumers/what-is-xbrl/) — 解释 XBRL（财报的机器可读格式）是什么
- [Seeking Alpha — 财报解读文章](https://seekingalpha.com/) — 看专业分析师怎么解读财报（注意：观点可能有偏见）
- [Fool.com — Earnings Call Transcripts](https://www.fool.com/earnings-call-transcripts/) — 免费 EC transcript 源

## Reference (工作时查阅)

- [edgartools 文档](https://edgartools.readthedocs.io/) — 我们用的 Python 库，理解它能做什么边界
- [SEC EDGAR API 文档](https://www.sec.gov/edgar/sec-api-documentation) — 底层 API 的能力边界

## Cross-holding data sources (新增 — Lessons 0006-0009)

### 13F 机构持仓 (Lesson 0006)
- [WhaleWisdom](https://whalewisdom.com/) — 13F 汇总 + 13D/G 追踪，免费层可见 Top 机构持仓。验证我们的 13F 采集结果
- [Dataroma](https://www.dataroma.com/) — Superinvestors 的 13F 持仓变动可视化。免费，看真实数据学习"一个机构持有多只股票"的展示方式
- [13F 视频教程 — Everything Money](https://www.youtube.com/results?search_query=how+to+use+13f+filings) — 讲清楚 13F 能做什么、不能做什么、45 天滞后意味着什么 (~12 min)

### Form 4 内部人交易 (Lesson 0007)
- [OpenInsider](http://openinsider.com/) — 最好的内部人交易数据网站，免费。直接搜索 CVNA/KMX/AN，看实际 Form 4 数据长什么样
- [Form 4 视频 — Investor's Business Daily](https://www.youtube.com/results?search_query=form+4+insider+trading) — Form 4 基础解读 (~8 min)

### 13D/G Activist (Lesson 0008)
- [13D Monitor](https://13dmonitor.com/) — Schulte Roth & Zabel 出版，行业最专业的 activist 追踪 newsletter
- [Aswath Damodaran — Activist Investing](https://pages.stern.nyu.edu/~adamodar/) — NYU Stern 估值教授，公开课讲 activist 什么时候会介入、为什么 (~20 min)
