# Notas de Enseñanza

## Preferencias del usuario
- **Objetivo de aprendizaje**: conocimiento del dominio (SEC, lógica de monitorización de inversión estratégica), no programación
- **Nivel**: poder explicar a un colega del departamento de inversión estratégica por qué cada pieza se hace así
- **Punto de partida**: manejo fluido de código, sin experiencia en SEC / análisis de estados financieros
- **Estilo**: principalmente en chino, con ejemplos reales (Carvana como prioridad)
- **Ritmo**: lecciones cortas, un concepto por vez, con quiz

- PRD v2.0 finalizado (`/Users/liuming/sec/PRD-competitor-intel-mvp-v2.md`)
- Competidores: Carvana / Carmax / AutoNation / 优信 (UXIN) / 汽车之家 (ATHM)

## Notas sobre el estilo de enseñanza

- El quiz debe tener un diseño "contraintuitivo": si la respuesta es demasiado obvia, no hay práctica de recuperación
