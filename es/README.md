Esta es la **versión en español** de un curso de 9 lecciones sobre cómo monitorear empresas públicas estadounidenses a través de sus presentaciones ante la SEC (10-K, 10-Q, 8-K, Form 4, 13F, 13D/G) para inteligencia competitiva.

## Cómo usar este curso

1. **Empieza aquí**: [`lessons/0001-sec-filing-101.html`](./lessons/0001-sec-filing-101.html) — introducción a 10-K, 10-Q, 8-K
2. **Sigue el orden recomendado**: 0001 → 0002 → 0003 → 0004 → 0005 → 0006 → 0007 → 0008 → 0009
3. **Ten a mano la hoja de referencia**: [`reference/sec-form-types.html`](./reference/sec-form-types.html) es imprimible
4. **No te saltes los cuestionarios** — están diseñados para detectar errores comunes de juicio

## Acerca de este curso

- **Objetivo**: Enseñar el **conocimiento del dominio** detrás del uso de las divulgaciones de empresas públicas estadounidenses para inteligencia competitiva
- **Audiencia**: Analistas de inversión estratégica, equipos de due diligence, investigadores de M&A, cualquiera que monitoree competidores de empresas públicas
- **Formato**: 9 archivos HTML autocontenidos, ~10-15 min cada uno, 3 preguntas de cuestionario por lección
- **Ejemplos**: Carvana (CVNA), Carmax (KMX), AutoNation (AN), Uxin (UXIN), AutoHome (ATHM)
