# SEC Lessons — Japanese Version

> **🌐 ようこそ / Welcome / Bienvenue / Willkommen / Bienvenido / 환영합니다 / 欢迎**

This is the **Japanese (日本語)** version of a 9-lesson course on monitoring US public companies through SEC filings (10-K, 10-Q, 8-K, Form 4, 13F, 13D/G) for competitive intelligence.

**注意 (Note)**: This README is in English as a placeholder. The lessons and supporting files have been translated into 日本語. If you are a native Japanese speaker and would like to translate this README, please see [CONTRIBUTING.md](./../../CONTRIBUTING.md).

## 使い方 (How to use this course)

1. **ここから始めてください**: [`lessons/0001-sec-filing-101.html`](./lessons/0001-sec-filing-101.html) — 10-K, 10-Q, 8-K の入門
2. **推奨順序**: 0001 → 0002 → 0003 → 0004 → 0005 → 0006 → 0007 → 0008 → 0009
3. **チートシートを手元に**: [`reference/sec-form-types.html`](./reference/sec-form-types.html) は印刷可能
4. **クイズはスキップしない** — 一般的な判断ミスを見つけるための設計です
