# SEC Lessons — English Version

> **🌐 Welcome / Bienvenue / Willkommen / ようこそ / 환영합니다 / 欢迎**

This is the **English** version of a 9-lesson course on monitoring US public companies through SEC filings (10-K, 10-Q, 8-K, Form 4, 13F, 13D/G) for competitive intelligence.

## How to use this course

1. **Start here**: [`lessons/0001-sec-filing-101.html`](./lessons/0001-sec-filing-101.html) — the 10-K, 10-Q, 8-K intro
2. **Follow the recommended order**: 0001 → 0002 → 0003 → 0004 → 0005 → 0006 → 0007 → 0008 → 0009
3. **Keep the cheat sheet handy**: [`reference/sec-form-types.html`](./reference/sec-form-types.html) is printable
4. **Don't skip the quizzes** — they're designed to surface common misjudgments

## About this course

- **Goal**: Teach the **domain knowledge** behind using US public-company disclosures for competitive intelligence
- **Audience**: Strategic-investment analysts, due-diligence teams, M&A researchers, anyone monitoring public-company competitors
- **Format**: 9 self-contained HTML files, ~10-15 min each, 3 quiz questions per lesson
- **Examples**: Carvana (CVNA), Carmax (KMX), AutoNation (AN), Uxin (UXIN), AutoHome (ATHM)
