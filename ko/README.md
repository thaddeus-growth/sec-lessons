# SEC Lessons — Korean Version

> **🌐 환영합니다 / Welcome / Bienvenue / Willkommen / Bienvenido / ようこそ / 欢迎**

This is the **Korean (한국어)** version of a 9-lesson course on monitoring US public companies through SEC filings (10-K, 10-Q, 8-K, Form 4, 13F, 13D/G) for competitive intelligence.

**참고 (Note)**: This README is in English as a placeholder. The lessons and supporting files have been translated into 한국어. If you are a native Korean speaker and would like to translate this README, please see [CONTRIBUTING.md](./../../CONTRIBUTING.md).

## 사용 방법 (How to use this course)

1. **여기서 시작**: [`lessons/0001-sec-filing-101.html`](./lessons/0001-sec-filing-101.html) — 10-K, 10-Q, 8-K 입문
2. **권장 순서**: 0001 → 0002 → 0003 → 0004 → 0005 → 0006 → 0007 → 0008 → 0009
3. **치트시트 가까이 두기**: [`reference/sec-form-types.html`](./reference/sec-form-types.html) 인쇄 가능
4. **퀴즈는 건너뛰지 마세요** — 흔한 판단 실수를 드러내도록 설계됨
