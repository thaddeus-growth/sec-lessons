# 자료 (Resources)

## Primary (신뢰도 최상, 여기서부터 시작)

### SEC 공식
- [SEC EDGAR — How to Research Public Companies](https://www.sec.gov/edgar/about) — SEC 공식 EDGAR 입문 가이드
- [SEC Beginners Guide to Financial Statements](https://www.sec.gov/about/reports-publications/investor-publications/beginners-guide-financial-statements) — SEC가 작성한 재무제표 읽기 입문(비재무 배경 대상)
- [EDGAR Form Types](https://www.sec.gov/files/forms-listed.pdf) — 모든 SEC form 유형의 완전 목록
- [How to Read a 10-K](https://www.sec.gov/files/reada10k.pdf) — SEC 공식 10-K 읽기 가이드
- [How to Read an 8-K](https://www.sec.gov/files/reada8k.pdf) — SEC 공식 8-K 읽기 가이드

### 전략적 투자 / 경쟁사 분석
- [CFI — Competitor Analysis Framework](https://corporatefinanceinstitute.com/resources/management/competitor-analysis/) — 경쟁사 분석의 고전 프레임워크
- [Aswath Damodaran — Valuation & Financial Statement Analysis Course](https://pages.stern.nyu.edu/~adamodar/) — 월가에서 가장 유명한 가치평가 교수, 모든 강의 자료 무료
- [Investopedia — Financial Statement Analysis Basics](https://www.investopedia.com/articles/basics/06/financialreporting.asp) — SEC 공식 가이드보다 더 쉽게 설명

### 중고차 산업 전용
- [Carvana Investor Relations](https://investors.carvana.com/) — Carvana의 IR 페이지를 직접 확인하여 상장 기업이 투자자와 소통하는 방식을 이해
- [NADA — Auto Retail Industry Data](https://www.nada.org/nada/industry-analysis) — 미국 자동차 딜러 협회의 산업 데이터

## Secondary (유용한 심화 자료)

- [XBRL US — What is XBRL?](https://xbrl.us/home/consumers/what-is-xbrl/) — XBRL(재무제표의 기계 판독 가능 형식) 설명
- [Seeking Alpha — Earnings Interpretation Articles](https://seekingalpha.com/) — 전문 애널리스트의 실적 해석 기사를 확인(의견에 편향이 있을 수 있음)
- [Fool.com — Earnings Call Transcripts](https://www.fool.com/earnings-call-transcripts/) — 무료 EC transcript 소스

## Reference (업무 중 참고)

- [edgartools documentation](https://edgartools.readthedocs.io/) — 우리가 사용하는 Python 라이브러리, 가능한 작업 경계 이해
- [SEC EDGAR API documentation](https://www.sec.gov/edgar/sec-api-documentation) — 기본 API의 기능 경계

## 교차 보유 데이터 소스 (신규 — Lessons 0006-0009)

### 13F 기관 보유 (Lesson 0006)
- [WhaleWisdom](https://whalewisdom.com/) — 13F 집계 + 13D/G 추적, 무료 티어에서 Top 기관 보유 확인 가능. 우리의 13F 수집 결과 검증에 활용
- [Dataroma](https://www.dataroma.com/) — Superinvestor의 13F 보유 변동 시각화. 무료, 실제 데이터를 통해 "한 기관이 여러 종목을 보유하는" 방식 학습
- [13F Video Tutorial — Everything Money](https://www.youtube.com/results?search_query=how+to+use+13f+filings) — 13F로 할 수 있는 것과 없는 것, 45일 지연이 의미하는 바를 명확히 설명 (~12분)

### Form 4 내부자 거래 (Lesson 0007)
- [OpenInsider](http://openinsider.com/) — 최고의 내부자 거래 데이터 웹사이트, 무료. CVNA/KMX/AN을 직접 검색하여 실제 Form 4 데이터 형태 확인
- [Form 4 Video — Investor's Business Daily](https://www.youtube.com/results?search_query=form+4+insider+trading) — Form 4 기초 해석 (~8분)

### 13D/G Activist (Lesson 0008)
- [13D Monitor](https://13dmonitor.com/) — Schulte Roth & Zabel 발행, 업계에서 가장 전문적인 activist 추적 뉴스레터
- [Aswath Damodaran — Activist Investing](https://pages.stern.nyu.edu/~adamodar/) — NYU Stern 가치평가 교수, 공개 강의에서 activist가 언제 개입하는지, 왜 그러한지 설명 (~20분)
