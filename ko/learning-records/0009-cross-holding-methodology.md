# Learning Record 0009 — 교차 보유 분석 방법론

**Date**: 2026-06-17
**Lesson**: 0009-cross-holding-methodology.html
**Status**: 완료

## 핵심 인사이트

1. **페이지당 하나의 구체적 투자 질문에 답하기**는 교차 보유 보고서의 핵심 설계 철학. IHS P4 보고서 10페이지 = 10개 질문. 데이터 나열 ≠ 보고서.
2. **페이지 간 교차 검증이야말로 진정한 통찰**. 4가지 고전 시나리오:
   - 확인적(P2 대규모 보유 + P3 추가 매수 + P5 Top Buyer) = 강한 신호
   - 모순적(P2 대규모 보유 + P3 소폭 매도 + P6 Top Seller 아님) = 재조정 가능성
   - 위험(P4 activist + P7 신규 진입) = 높은 경고
   - 전환(P8 수동 자금 증가 + P9 가치/성장 로테이션) = 산업 생명주기 변화
3. **올바른 보고서 읽기 순서**: 30초 속독 → 2분 추세 → 5분 심층 → 15분 연구. Dashboard 설계는 이 리듬을 지원해야 함(개요 카드를 가장 눈에 띄게, 심층 데이터는 탭 안에).
4. **6가지 설계 원칙**:
   - 페이지당 하나의 질문에 답
   - 페이지 하단에 Commentary 공백(애널리스트가 판단을 적는 곳)
   - 테이블 열은 업계 공통 언어와 정렬(Style / Turnover / Peer Average 빠짐없이)
   - 순위 테이블은 반드시 순위 지표 명시
   - 차트와 테이블은 동반(파이 차트는 분포, 막대 차트는 자금 흐름)
   - 면책 조항은 반드시 가시화(문서 끝이 아닌 페이지 하단)
5. **Top Buyers/Sellers 단방향 흐름 = 99% 버그**. 정상적으로는 양방향(어떤 이는 사고 어떤 이는 판다). 일반적 원인: 기간 간 비교 오류, 필드 매핑 오류, 데이터 소스 결손. **자체 점검 리스트에 "Buyers와 Sellers가 모두 데이터를 가지는지" 반드시 추가**.
6. **기간 간 비교는 시가총액(value delta)이 아닌 주식 수(shares delta) 기준**. 시가총액 변화는 주가 변동에 오염됨. Top Buyers는 "능동 추가 매수" 또는 "주식 수 변화"로 정렬해야.
7. **Carvana 2026Q1 실전 분석**: Vanguard/BlackRock 보유 변동 없음(수동 인덱스 자금 안정) + Baupost $480M 신규(가치 투자자 재진입) + activist 부재 = "인덱스 안정 + 가치 재평가 + activist 간섭 없음" → 약간 긍정적 상태.
8. **산출 품질 10항목 자체 점검 리스트**: 단위, 순위 지표, institution_name, delta_pct 경계, Buyers/Sellers 대칭, 인접 기간, 면책 조항, 시차, Item 4 리뷰, 중국 ADR "적용 불가" 표기.
