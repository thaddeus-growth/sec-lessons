## 핵심 인사이트

1. **Section 16의 적용 경계**: <strong>미국 등록 증권</strong>을 보유한 내부자에게만 적용. 중국 ADR(UXIN/ATHM)의 임원은 보통 VIE 단계 지분을 보유 → 적용되지 않음. 따라서 "데이터 없음" ≠ "수집 실패", 구조적 "적용 불가".
2. **세 가지 테이블의 차이**: Form 3(최초 신고, 1회성) / Form 4(매 거래, 2 영업일, 가장 중요) / Form 5(연간 집계, 면제 거래).
3. **Table I vs Table II**: Table I = 직접 보유; Table II = 옵션/파생상품. M(행사) + F(세금 매도) = 동반 작업이며, 약세 신호가 아님.
4. **거래 코드 가중치 설계**:
   - P(능동 매수) = 1.0(강한 신호)
   - S(능동 매도) = 1.0
   - F(세금 매도) = 0.2(미세한 신호 유지)
   - G(증여) = 0.3
   - A(부여) = 0.0(능동 거래 미산정)
   - M(행사) = 0.0(행사 후 매도만 참고)
5. **역할 가중치**: CEO/CFO = 2.0(가장 민감) / 10% Owner = 1.5 / Director = 1.0 / EVP/SVP/VP = 0.8.
6. **10b5-1 계획은 최대 오판 원인**: 임원 사전 설정 거래 계획은 다량의 페어링 Code F/S를 생성. **판단 시 반드시 10b5-1 계획의 신고일 확인 필수**(8-K Item 8.01 또는 DEF 14A에 있음), 거래 자체가 아님.
7. **Form 144는 "예고편"**: 사전 통지이며 사후 공시가 아님. "90일 내 매도 예정" ≠ "이미 매도함". 다수의 Form 144 연속 제출 = 지속적 매도 신호.
8. **중국 ADR 이상 경고 규칙**: UXIN/ATHM이 갑자기 Form 4를 제출하면 → 이것은 극히 중요한 신호(희귀 이벤트), 시스템은 적색 경보를 발동해야 함.
